Linters

The repo is set up to use prospector, a tool that combines several python code-checkers including pylint.

prospector is run for every commit pushed to github via landscape.io; see CI page for more info.

Installation

./scripts/install-lint

Usage

prospector

For info on usage through a text editor see text editor setup.

Configuration

  • Prospector: .landscape.yml
    • Thus named because it also configures landscape.io runs
    • More info: prospector profiles
  • Pylint: .pylintrc
    • More info: pylint docs
