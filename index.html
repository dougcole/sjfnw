Contributing

Follow the table of contents to get a local development environment set up, starting with installation and setup.

The resources page has links with more info on the various languages/platforms/technologies that are used. If you are new to Django, definitely look through the Django docs.

Check issues labeled "ready" to see what needs doing. Post a comment on the issue if you need clarification. See issue tracking for more info.

If you notice anything missing/wrong in the docs, please submit a PR!
