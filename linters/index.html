Project is set up for pylint and eslint, syntax & style checkers for python and javascript.

Installation
  • pylint:
    • sudo apt-get install pylint or pip install pylint (details here)
    • make sure you have v1.4+ (do pip list to see, pip -U pylint to upgrade)
    • pip install pylint-django - plugin that makes pylint more django friendly
  • eslint: (optional)
    • You'll need npm first: sudo apt-get install node + sudo apt-get install npm or see their docs for instructions.
    • npm install -g eslint See this page if you get a permissions error.
Usage - in text editor

See [[text editor setup|text-editor-setup]] for more info.

Usage - CLI
  • To lint the whole project: ./scripts/lint.sh
  • That will output results to the console. If you want to save results to a file, you can do ./scripts/lint.sh > filename.txt
